# HRMS Lite

A lightweight Human Resource Management System built with FastAPI + React.

**Live URL:** https://inspiring-sorbet-8d8854.netlify.app/

> **Note:** The backend is hosted on Render's free tier. The first load may take **30–60 seconds** due to cold start — please wait a moment for it to wake up.


## Tech Stack

- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** FastAPI, SQLAlchemy
- **Database:** PostgreSQL
